# 使用storybook 开发vue组件

> vue组件开发

``` bash
#克隆项目
git clone https://github.com/yukari521/StorybookVueComponents.git

#安装依赖
npm install

#本地运行
npm run storybook

#将组件预览打包成静态资源
npm run build

# 全局安装storybook
npm i -g @storybook/cli

```

[Storybook for Vue](https://storybook.js.org/docs/guides/guide-vue/)
